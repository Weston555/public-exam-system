# "公考进阶"个性化学习路径推荐系统 - 需求规格说明

## 项目概述

本系统名为"公考进阶"个性化学习路径推荐系统，旨在通过构建"考试大纲—知识点—题目—难度—目标成绩"的一体化模型，为公考考生提供可解释、可配置的个性化学习路径，解决学习路径同质化、练习与大纲脱节、复习缺乏针对性等痛点。

## 核心功能需求

### 学员端功能

#### 1. 账户与目标配置
- 用户注册/登录
- 目标成绩设置（目标分数、考试日期）
- 学习时间配置（每日学习时长）
- 个人资料管理

#### 2. 基线诊断
- 自动生成诊断试卷（覆盖所有知识点）
- 在线答题功能
- 实时保存答题进度
- 自动判分与能力评估

#### 3. 个性化学习路径生成
- 基于诊断结果生成学习计划
- 支持7天/14天/30天计划
- 按知识点优先级排序
- 包含学习理由解释

#### 4. 针对性练习
- 按知识点专项练习
- 难度自适应调整
- 练习历史记录
- 实时反馈与解析

#### 5. 模拟测验
- 全真模拟考试
- 计时功能
- 成绩分析与对比

#### 6. 学习进度追踪与提醒
- 每日学习任务提醒
- 进度完成度统计
- 学习效果可视化
- 错题本与定期复习

### 管理员端功能

#### 1. 考试大纲与知识点树管理
- 大纲结构维护
- 知识点树形结构管理
- 知识点权重配置
- 预计学习时间设置

#### 2. 题库与学习资源维护
- 多题型支持（单选、多选、判断、填空、简答）
- 题目CRUD操作
- 题目难度设置
- 知识点标签关联

#### 3. 推荐策略与实验参数配置
- 学习路径生成算法参数
- 难度自适应规则
- 优先级计算公式
- A/B测试配置

#### 4. 用户权限治理
- 用户角色管理
- 权限分配
- 用户状态管理

#### 5. 数据质量监测
- 学习数据统计
- 题库质量分析
- 系统使用情况监控

## 技术需求

### 前端技术栈
- Vue 3 + Composition API
- Vite 构建工具
- Vue Router 路由管理
- Pinia 状态管理
- Element Plus UI组件库
- ECharts 图表库
- Axios HTTP客户端

### 后端技术栈
- Python FastAPI
- SQLAlchemy ORM
- MySQL 数据库
- JWT 身份认证
- Pydantic 数据验证

### 非功能需求
- 响应式设计，支持PC端
- RESTful API设计
- 数据安全性与隐私保护
- 高可用性与性能优化
- 可扩展性架构设计

## 数据模型

### 核心实体关系
- 用户(User) - 目标(Goal)：1:N
- 用户(User) - 知识点掌握状态(UserKnowledgeState)：1:N
- 知识点(KnowledgePoint) - 题目(Question)：N:M
- 试卷(Paper) - 题目(PaperQuestion)：1:N
- 考试(Exam) - 作答记录(Attempt)：1:N
- 作答记录(Attempt) - 答案(Answer)：1:N
- 用户(User) - 错题(WrongQuestion)：1:N
- 目标(Goal) - 学习计划(LearningPlan)：1:N
- 学习计划(LearningPlan) - 计划项(PlanItem)：1:N

## 接口要求

### 认证相关
- POST /auth/register - 用户注册
- POST /auth/login - 用户登录
- GET /users/me - 获取当前用户信息

### 学员端接口
- 目标管理：CRUD操作
- 诊断考试：生成、答题、提交判分
- 学习计划：生成、查看、完成标记
- 练习接口：按条件获取题目、提交答案
- 数据统计：进度、趋势、分析接口

### 管理员端接口
- 知识点树：树形结构CRUD
- 题库管理：题目CRUD
- 试卷管理：组卷、发布
- 考试管理：创建、发布、监控
- 用户管理：权限、状态管理
- 数据导出：脱敏数据导出

## 验收标准

### 功能验收
- [ ] 学员可完成完整学习闭环：注册→目标设置→诊断→学习计划→练习→模考→数据分析
- [ ] 管理员可管理完整内容：知识点→题库→试卷→考试→用户→数据监控
- [ ] 系统具备可解释性：学习路径生成有理由说明
- [ ] 支持难度自适应：练习题目难度根据掌握度调整

### 性能验收
- [ ] 页面加载时间 < 3秒
- [ ] API响应时间 < 1秒
- [ ] 支持并发用户数 > 100
- [ ] 数据库查询优化

### 质量验收
- [ ] 单元测试覆盖率 > 70%
- [ ] 代码规范检查通过
- [ ] 安全漏洞扫描通过
- [ ] 用户体验评估通过
